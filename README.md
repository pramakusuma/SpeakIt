## Features

- Progress tracker
- Play word/sentence example
- Speech recognition
- Speech accuracy

## Requirements

- iOS 15.0+
- Xcode 15.2

## Installation

### Xcode install

#### Manually
1. Download and drop ```SpeakIt.xcodeproj``` in Xcode.  
2. Run the app
